# Prompts Used During Development

The following prompts were used to assist in developing the Cash-Flow Salary & Expense Tracker.

---

## Prompt 1

Create a responsive Salary and Expense Tracker using HTML, CSS, and JavaScript.

---

## Prompt 2

Design a modern dashboard UI with salary cards, expense cards, responsive layout, and dark theme.

---

## Prompt 3

Create JavaScript logic for:

- Add Salary
- Add Expense
- Remaining Balance
- Validation

---

## Prompt 4

Store application state using LocalStorage and automatically restore data after page refresh.

---

## Prompt 5

Create delete functionality for expenses that updates:

- DOM
- LocalStorage
- Remaining Balance
- Chart

---

## Prompt 6

Integrate Chart.js Doughnut Chart showing:

- Remaining Balance
- Expense Breakdown

Update dynamically whenever data changes.

---

## Prompt 7

Generate PDF report using jsPDF including:

- Salary
- Expenses
- Remaining Balance
- Date
- Expense Table

---

## Prompt 8

Integrate real-time Currency Conversion using Exchange Rate API.

Supported currencies:

- INR
- USD
- EUR
- GBP

---

## Prompt 9

Implement threshold alert when remaining balance becomes less than 10% of total salary.

Display:

- Red balance
- Warning banner

---

## Prompt 10

Improve UI responsiveness for desktop, tablet, and mobile devices.

---

## Prompt 11

Improve code readability using:

- Modular functions
- Clear variable names
- Reusable helper functions

---

## Prompt 12

Optimize rendering by updating only necessary DOM elements and reusing existing Chart.js instances.

---
